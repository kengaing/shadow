# What is this for?

Get perfect api's

# Installation
`npm i shadow --save`

Example:
```
const shadow = require('shadow');

shadow({
    shadow_type: 'soft'
    padding: false
});
```

# Options

Shadow 2 support otions, both of which are optional:

* *shadow_type* - _hard | soft_ (Default to soft)
* *padding* - _boolean_ (Default to false)
